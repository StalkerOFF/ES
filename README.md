# Песок Трекер - Инструкция по установке

## Описание
Сайт для отслеживания задач новых сотрудников ("песка") в стиле темного Liquid Glass.

## Функционал
- Создание новых списков песка с выбором РОПа, даты, типа работы (офис/удаленка) и списка сотрудников
- Таблица с задачами, где можно отмечать выполненные пункты галочками
- Разные наборы столбцов для офиса и удаленки
- Автоматическое скрытие столбца "Добавление на Libra" для определенных РОПов
- Дополнительный пункт "Проконтролировать технику в классе песка" для офиса
- Редактирование существующих списков
- Сохранение всех данных в Supabase

## Настройка Supabase

### Шаг 1: Войдите в панель управления Supabase
Перейдите по ссылке: https://wnvjebxkfjqnyyzpsszc.supabase.co

### Шаг 2: Создайте таблицу
1. Перейдите в раздел **SQL Editor** в левой панели
2. Нажмите **New Query**
3. Скопируйте содержимое файла `supabase_schema.sql` и вставьте его в редактор
4. Нажмите **Run** для выполнения SQL-запроса

### Шаг 3: Проверьте создание таблицы
1. Перейдите в раздел **Table Editor**
2. Убедитесь, что таблица `sand_lists` создана

## Использование

### Создание нового списка
1. Нажмите кнопку "+ Новый песок" в правом верхнем углу
2. Выберите РОПа из выпадающего списка
3. Выберите дату в календаре
4. Выберите тип работы (Офис или Удаленка)
5. Введите список фамилий и имен сотрудников (каждый с новой строки)
6. Нажмите "Сохранить"

### Отметка выполненных задач
- Кликайте по ячейкам таблицы для установки/снятия галочек
- Данные сохраняются автоматически в Supabase

### Редактирование списка
1. Нажмите кнопку "Редактировать" на карточке списка
2. Измените необходимые поля
3. Нажмите "Сохранить"

## Особенности столбцов

### Для офиса:
- Создать аккаунты Битрикс
- Добавление на Libra (кроме РОПов: Макаренко, Власов, Сайдаралиев, Шабуров)
- Создать ключи в росгварде
- Налить холодку
- Отключить акки невышедших сотрудников
- Проконтролировать технику в классе песка (отдельно внизу)

### Для удаленки:
- Создать аккаунты битрикс
- Добавление на Libra (кроме РОПов: Макаренко, Власов, Сайдаралиев, Шабуров)
- Налить холодку
- Настроить технику
- Отключить акки невышедших сотрудников
